API Documentation for ‘data_processor’¶

Create a simple data pipeline using Yellow Taxis trip data

  • Here we specify the documented module name:

This module provides analysis of NYC Yellow Taxi data.

data_processor.calculate_monthly_average(df)

Calculates average trip length per month. :param df: dataframe :return: Average trip length per month.

data_processor.clean_data(df, index)

Cleans up data so that only relevant data is processed during calculations. :param df: Data Frame to analyse :param index: Month of trip data :return: Cleaned up data frame

data_processor.initialize()

Enable logging to a file and to console. :return:

data_processor.main()

Processes data source files (in CSV format) in a loop. :return:

data_processor.read_trip_data(url)

Loads CSV file in chunks of 100 MB and return dataframe. :param url: url for loading csv data. :return: dataframe

data_processor.rolling_average(df)

This function calculates rolling average per month. :param df: dataframe :return: Rolling average per month
